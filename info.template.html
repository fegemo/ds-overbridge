Data

The data used to create this visualization.

Data about the movies (characters etc.)

The data regarding the movies info was grabbed from the Star Wars API project. It does not provide full files for each entity, so I made a lot of requests to fetch all of them and then assembled the responses in the files:

  • planets.json (61 planets)
  • people.json (87 people)
  • species.json (37 species)
  • starships.json (37 starships)

The API has even more data, about the movies (7 so far) and vehicles (39), but they were not used here.

Data about the galaxy (its geography)

The geographic data was acquired from the Star Wars Galaxy Map and comprises 5 sets of topography in the geojson format, which became layers on the map: grid, planets, hyperspace (not used), region, sector.

To reduce its size, the geographic data was simplified using mapshaper using the Visvalingam method with weighted area. That yielded 44% size reduction with only a small perceived loss.

Regarding planets, the Star Wars Galaxy Map has about 2,068 planets with their coordinates, comprising all of canon and "extended universe" locations. However, as the movies database had information about only 61 (canon) planets, only those were considered.

Joining the data

Only planets that exist both on the geographic database and on the movies database were considered. To join their data, two fields called "name" and "name_web" from the planets layer from the geography database were tentatively matched with a field called "name" on the planets.json from the movies database.

Death Star's Overbridge

Enter the III Death Star's Overbridge and decide what to destroy next.

About

What if the 3rd Death Star was actually constructed? Wear Lord Vader's boots and decide, from its control room (the Overbridge), which planets to destroy with the death ray next. To aid in the decision, the Overbridge allows the visualization of:

  1. The galaxy, its regions, sectors and most notable planets
  2. What are the climates and terrain types of the selected planets
    • Allows understanding what sort of troops to send down, if necessary
    • Allows seeing which types of climates/terrains might cease existing if the selected planets are indeed destroyed
  3. What are the languages spoken on those planets and the number of individuals on the selected planets speaking each one
  4. Some watched individuals that might have their home planets (and lives) destroyed by the next move, grouped by their species and color-coded by gender
  5. The types of starships those watched individuals know how to pilot
    • Some starships might become useless if all of its pilots are lost

Movies Data

The planets, movies, people, species and all non-geographic data used by this project comes from the SWAPI - the Star Wars API, made and maintained by Paul Hallet. The SWAPI is licensed under a BSD license.

Paul Hallet built and maintains SWAPI by himself and needs support with its costs. If you like this or his project, please consider donating for him to keep the API working forever, as it is a great resource for learning about Web/AJAX/APIs etc., while also holding precious data.
